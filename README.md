# Play Tennis – Decision Tree (ID3)

This repository demonstrates the implementation of the Decision Tree ID3 algorithm
using the Play Tennis (Weather) dataset.

## Features
- Label Encoding of categorical data
- Decision Tree using Entropy (ID3)
- Model evaluation using standard classification metrics
- Prediction on new unseen data
- Decision Tree visualization

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
